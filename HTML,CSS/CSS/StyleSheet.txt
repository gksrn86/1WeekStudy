Style에는 세가지의 스타일 시트가 있습니다.
첫 번째 External style sheet로서
css파일을 직접 생성해서 태그마다 스타일을 지정해줘서 css파일과
연동해서 사용하는 방법으로 가장 많이 이용하는 방법!
style.css 안에
p{}
h1{} 같은 방법

두 번째Internal style sheet는
현재 사용하는 html 파일 속에 <head></head>태그 안에
<style></style>태그를 생성해서 그 안에 태그에 대해서 스타일을 지정해주는 것으로
스타일 지정할 것이 적다면 이용이 좋으나 잘 사용하지 않는 방법

세 번째 태그 안에 즉 <h1 style=""></h1>을 통해서 스타일을 주는 방법으로
이 방법도 바디 안에 많은 글자가 들어가면 어떤 스타일을 줬는지 찾는 부분이 쉽지 않기
때문에 잘 사용하지 않습니다.
